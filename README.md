# Running
```
go build
./kivaranger
```

Arrow keys/hjkl to navigate \
`Home` or `g` to go to the top \
`End` or `G` to go to the bottom \
`M` to go to the middle \
q to quit \
F1 to open in nano \
Delete to delete file(s) \
Spacebar to select files \
`A` to flip selection in folder (select all files) \
`a` to rename a file
